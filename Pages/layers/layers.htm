The importance of layers

Instead of one complex image imagine it broken down into seperate parts with each part on its own transparent layer. There are different types of layers and one of the most important is the adjustment layer.

Looking down you would see just one image but it allows us to make adjustments to each single layer without effecting the others. If you make changes to one single layer then pixels are destroyed and you may not get them back. You'd probably want to start over again. You can lock layers so they can't be edited by accident.

 

The image in the topmost layer in PS is appropriately on top of all the other images.

If the name of the layer is in italics then that layer is not transparent. To change this double click and.select area with any selection tool and either:

  1. use the Eraser tool
  2. drag the transparancy slider in the layer's palette
  3. use a layer mask - you'll find a button at the bottom of the layers palette

The eye turns visibility on and off and you can give each layer a colour and also group layers together.

Another useful feature is that you can merge layers. Say for example your base or background layer is not complete you can add another layer with the missing bits on and then merge them.

 

The 7 main layer types

There are different layer types and you choose according to what you want to do.

1. Image Layer

It's a good idea to duplicate your background or base layer. (See the importance of layers)

You can drag down the original layer to the new layer icon or use Ctrl + J keyboard shortcut.

To add another image as a layer you can open it, Select > All and then copy it and then paste it onto the other image

2. Adjustment Layer

This is probably the most often used layer as you can make all kinds of adjustments to an image such as add gradients, brightnness, colour changes, intensity, filters etc etc.

It's important to realise that the adjusmrnt made in this layer will affect all the layers below it, not just that layer. So you may need to arrange the layers by dragging them up or down.

Layer > New adjustment layer

3. Fill Layers

This layer enables us to create artificial colours and gradients that didn't exist in the original.

Used with blending modes like screen or overlay make this layer powerful

Layer > New Fill layer or

4. Shape Layer

Thes are vector shames which can be enlarged and used to make logos etc.

5. Type (Text ) layer

You can create cool text effects in photshop that may make it better the Word for making invitations for example.

6. Smart Object Layer

This layer preserves all the layers, meta data about an image and allows you to adjust or edit them. Using the history menu you can see what's been done to the image (even if created in Illustrator) and can turn adjustments on and off.

7. Video Layers

If you have a video and you want to import certain frames from the video this layer type will do it for you.
